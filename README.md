HOW TO INSTALL?

Download "animated-artworks.mjs"
then put the file into this directory
C:\Users\YourName\AppData\Roaming\spicetify\Extensions
then run this command in your powershell:
"spicetify config extensions animated-artwork.mjs"
"spicetify apply"


then drop the file "animart-proxy.js"
then put the file into this directory
C:\Windows\System32

Before running the proxy, you must install ffmpeg first.
run powershell as administrator
install ffmpeg "winget install ffmpeg" in your powershell
After ffmpeg is ready to be installed, run the proxy.
"node animart-proxy.js"
